# Техническое задание

## Установка

1. Склонируйте репозиторий:

```sh
git clone https://github.com/maky3/texZadanie
cd texZadanie
```

2. Развернуть проект можно с помощью:

 ```sh
 docker compose up --build
 ```

3. Запустить тесты можно с помощью:

 ```sh
 php vendor/bin/phpunit
 ```
